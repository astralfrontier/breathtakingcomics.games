# Breathtaking Comics

This is the source code for the [Breathtaking Comics TTRPG website](https://breathtakingcomics.games/).
It uses the following tech:

- [Astro](https://astro.build) - for static site generation
- [MDX](https://mdxjs.com) - MDX, or expanded Markdown, for content
- [PicoCSS](https://picocss.com) - lightweight no-JS styling
- [Ramda](https://ramdajs.com) - for assorted functional coding
- [React](https://react.dev) - for front-end components

Individual heroes, villains, and other setting elements are in the `src/world` directory.
